# Coffee-Shop-Sales-Analysis-SQL
"End-to -end SQL analysis of coffee shop transactions, including data cleaning and KPI reporting"
This project analyzes transaction data from a coffee shop to identify sales trends, peak hours, and month-over-month growth.

Key Steps Taken
Data Cleaning: Formatted dates and times for time-series analysis.

Metric Calculation: Developed queries for Total Sales, Order Volume, and Average Transaction Value.

Advanced Analytics: Used Window Functions to compare daily performance against monthly averages

 Key Findings
Peak Traffic:** High-volume sales occur between **8:00 AM and 10:00 AM**.
Daily Performance:** Sales are consistently higher on **weekdays** than weekends.
  Growth:** Observed a steady **Month-over-Month (MoM)** increase in transaction volume.

 Technical Skills Demonstrated
Data Cleaning:** `STR_TO_DATE`, `ALTER TABLE`, and `UPDATE`.
Advanced Queries:** Window Functions (`LAG`, `OVER`), CTEs, and Subqueries.
Business Logic:** Conditional formatting using `CASE` statements.
